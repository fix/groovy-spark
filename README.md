this is a groovy implementation of spark

you may install it:

`echo "echo 'groovy https://raw.github.com/fix/groovy-spark/master/Spark.groovy "$@"' > /usr/bin/spark" | sudo sh `

Yes that easy, and you will always be up-to-date!
